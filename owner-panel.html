Integracja z GitHub nie jest skonfigurowana

Ustaw poniższe zmienne środowiskowe na serwerze, aby włączyć integrację z GitHub:

Zmienne środowiskowe
  • GITHUB_TOKEN=ghp_twój_token – Personal Access Token z uprawnieniami repo i pull_requests
  • GITHUB_REPO=owner/repo-name – nazwa repozytorium (np. uszefapromo-hub/HurtDetalUszefaQUALITET)
Bezpieczeństwo Token jest przechowywany wyłącznie jako zmienna środowiskowa serwera. Nigdy nie jest przekazywany do przeglądarki.

Konfiguracja

Ustawienia SMTP (Proton Mail Bridge)

Proton Mail Bridge Zainstaluj Proton Mail Bridge na serwerze i skonfiguruj następujące zmienne środowiskowe:
  • SMTP_HOST=127.0.0.1
  • SMTP_PORT=1025
  • SMTP_USER=twoj@protonmail.com
  • SMTP_PASS=bridge_password
  • SMTP_FROM=noreply@uszefaqualitet.pl
Status integracji System automatycznie wykrywa konfigurację SMTP przy każdym wysłaniu wiadomości.
  • Gdy SMTP_HOST nie jest ustawiony — wiadomość zostaje zapisana w kolejce
  • Po skonfigurowaniu Bridge — wiadomości są wysyłane natychmiast
